# Premier Opus Architects

A modern construction portfolio website built with TanStack Start, React Router, TypeScript, and Tailwind CSS.

## Quick Start

```bash
npm install
npm run dev      # Development server
npm run build    # Production build
npm run preview  # Preview production build
```

## Deployment

Deployed on Vercel with server-side rendering (SSR).

- **Live**: https://construction-site-yiez.vercel.app

## Tech Stack

- **Framework**: TanStack Start + React
- **Routing**: TanStack Router
- **Styling**: Tailwind CSS + Radix UI
- **Language**: TypeScript
- **Build**: Vite
- **Deployment**: Vercel
